# Event Management System

A Java-based web application for managing events.

## 👥 Team
* [dinsaradasin]
* [Sasindupathirana]

## 🛠️ Technologies Used
* **Backend:** Java
* **Frontend:** HTML, CSS
* **Build Tool:** Maven
* **Server:** Apache Tomcat
* **IDE:** Apache NetBeans

## 🚀 How to Run the Project
1. Clone this repository to your local machine:
   `git clone [https://github.com/dinsaradasin/Event-Management-System.git]`
2. Open **Apache NetBeans**.
3. Go to `File` > `Open Project` and select the cloned folder.
4. Right-click the project in NetBeans and select **Clean and Build** to download the Maven dependencies.
5. Click **Run** (or press F6) to deploy the application to your local Apache Tomcat server.

## 📂 Project Structure
* `src/main/java`: Contains all backend Java code (Models, Servlets, DAOs, Utils).
* `src/main/webapp`: Contains all frontend HTML/CSS files.
* `pom.xml`: Maven configuration and dependencies.
